Section 01

Strategic Vision

The ICA website today serves mostly as a static information page. This proposal outlines how to grow it into a working platform — one that supports program registration, family communication, and daily school operations in one place.

"The school website will serve as ICA's central hub for communications and programs — making it easier for parents to get information, improving event and program participation, and giving administration clear data on what's working."

Three core shifts define this transformation:

→ Shift 1

From Brochure to Platform

Manage program registrations, communications, and student and parent resources all in one place.

→ Shift 2

Less Repetitive Work

Social posts, event reminders, and email updates run on schedules — reducing workload.

→ Shift 3

Clear Performance Data

Every major component tracks real KPIs — page visits, event sign-ups, email opens — so decisions are based on data, not guesswork.

Social Media & Automation

Social Media & Automation

The website becomes the source of record for all school content. Social channels — Facebook, Instagram, YouTube, and LINE — act as the distribution layer while the site remains the central hub for school programs, updates, and parent information.

Publish to the website first, then share that content across channels, maintain consistent messaging, and track referral traffic back to the site.

Integration Tiers

Tier 1

Social Buttons & Embeds

Social follow buttons, homepage links, YouTube embeds, and share options on updates and news posts.

Tier 2

Scheduled Drafts

News and event updates can be prepared in advance, published on schedule, and reused across channels.

Tier 3

Unified Reporting

Track website traffic, social reach, and engagement with GA4 and native platform analytics.

Section 03

Core Program Microsites

Each major school program gets its own branded landing area under the main ICA domain. The Summer Game Dev Camp can be the first — and becomes the template for every camp, club, and program that follows.

Summer Game Dev Camp

Registration flow, project gallery, instructor profiles, FAQ, and social sharing. Fully tracked with KPIs.

AI & ML Camp

Python, data basics, and hands-on ML projects using Jupyter and Colab — 1–2 weeks.

Web Dev Workshop

HTML, CSS, JavaScript, and GitHub Pages. Students leave with a live portfolio site.

AP Cyber Prep Camp

Ethical hacking, network defense, and digital safety grounded in Taiwan critical infrastructure.

Senior Thesis Defense

Portal for students, parents, and evaluators with submission deadlines and defense scheduling.

Section 04

CS Program Microsite

The Computer Science and Computational Mathematics program at ICA runs on tools no other school in Taiwan uses — GitHub Codespaces, VS Code, Docker, Jupyter, Python, and a custom grading pipeline handling 500+ student repositories. That's a real differentiator, and it deserves its own page.

Active Repos

Industry-Standard Tools

Four-Year Curriculum Path

Algebra 2 → Computational Math → Coding → AP CS — a clear track families can see.

Student Portfolio Gallery

Published projects via GitHub Pages — real portfolios families and universities can browse.

Section 06

Reusable Program Framework

Beyond academics and camps, every school program — athletics, clubs, student government, international exchange — gets its own section, built from the same modular template.

Athletics Hub

Basketball roster, game schedule, score recaps, coaching profiles, and photo/video gallery. Results sync automatically.

Clubs Directory

Searchable list of all student clubs with meeting times, faculty sponsors, and a simple sign-up form.

Student Government

Meeting minutes, school initiatives, student polls, and a proposal submission channel.

International Exchange

Partner school directory, MOU listings, delegation archives — linked to Taiwan MOE.

Thesis Archive

Public archive of completed thesis projects. Students submit work and track defense schedules.

Events & Calendar

Unified school calendar with online registration and payment. No more paper forms.

Section 08

Administrative Efficiency

Every page that answers a question a parent or student would otherwise come to the office for is a real reduction in workload. These self-service modules deliver the highest return on effort.

Parent Self-Service Portal

Attendance, grades, test dates, fee schedules, and teacher contact forms — no office visit needed. Reduces repeat inquiries by 50–60%.

Emergency Notifications

One publish action updates the website, sends a newsletter, posts to LINE, and publishes on Facebook simultaneously.

Forms & Documents

Permission slips, medical forms, club sheets, and official documents — all downloadable and submittable online.

Monthly Dashboard

Internal analytics showing web traffic, email performance, social reach, and video views — auto-generated via GA4 and Looker Studio.

Section 10

Implementation Timeline

Each phase delivers working value before the next one starts. Phase 1 focuses on foundation and the Summer Game Dev Camp — an early win that proves the platform's value to administration before we scale.

Phase 1Months 1–2

Foundation & Pilot Launch

New website design and CMS setup. Summer Game Dev Camp microsite live. GA4 installed. Basic social media integration. LINE Official Account connected. Newsletter sign-up live.

Phase 2Months 3–4

CS Microsite & LMS Integration

CS Program Microsite launched with GitHub stats, curriculum map, and student portfolio gallery. Google Classroom links embedded. Clubs directory and athletics hub live. Parent Self-Service Portal active.

Phase 3Months 5–7

Automation & Family Engagement

Automated social workflows live. Emergency notification system connected. Bilingual content expanded. AI Camp and Web Dev Workshop microsites built. Monthly Looker Studio dashboard running.

Phase 4Month 8+

Full Platform

Senior Thesis Defense Portal live. International Exchange Hub connected to Taiwan MOE. Student Government portal. Full cross-channel reporting. Annual digital review and strategy refresh.

Recommended Tools & Stack

Recommended Tools & Stack

Built around tools that are free or low-cost at school scale, connect well with Google Workspace, and can be maintained by a small team without dedicated IT staff.

Analytics

Google Analytics 4

Free KPI tracking, event measurement, and page performance insights across all major pages.

Reporting

Looker Studio

Free Google dashboard tool for live reports and simple monthly review workflows.

Communication

LINE Official Account

Important for Taiwan parent communication, reminders, and school-wide announcements.

Hosting

GitHub Pages

Static hosting for program microsites, student-facing pages, and project showcase content.

Cost note: Google Analytics 4, Looker Studio, LINE Official Account, and GitHub Pages support a very low-cost launch model using tools ICA already knows or can adopt quickly.
